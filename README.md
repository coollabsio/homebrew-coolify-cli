# Homebrew Tap for [Coolify CLI](https://github.com/coollabsio/coolify-cli)

## Installation

```bash
brew install coollabsio/coolify/coolify
```

## Upgrade

```bash
brew upgrade coolify
```

## Uninstall

```bash
brew uninstall coolify
```
